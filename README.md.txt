CV_Projects
===========

A collection of computer vision projects implemented in Python using OpenCV and other related libraries. Each project demonstrates a practical application of computer vision techniques, including real-time video processing, object tracking, and geometric analysis.

Projects Included:
------------------
- Real-Time Angle Detector
  Detects and displays angles formed by joints or lines in real-time using video input.

- Object Tracker
  Tracks selected objects in a video stream using tracking algorithms like KCF or CSRT.

- Edge Detection Tool
  Applies various edge detection filters (Sobel, Canny) to live or static images.

- More to come

Technologies Used:
------------------
- Python
- OpenCV
- NumPy
- Matplotlib (optional)
- Dlib / Mediapipe (optional depending on project)

Getting Started:
----------------
1. Clone the repository:
   git clone https://github.com/yashringe/CV_Projects.git
   cd CV_Projects

2. Install dependencies:
   pip install -r requirements.txt

3. Run any project:
   python CV_angleTracker.py
